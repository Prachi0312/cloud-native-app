\# Cloud-Native Data Pipeline (Healthcare)



\## 📌 Overview

Built a scalable cloud-native data pipeline using Medallion Architecture (Bronze, Silver, Gold) for processing healthcare data.



\## 🚀 Key Achievements

\- Improved pipeline performance by 40%

\- Reduced manual reporting by 70%

\- Designed scalable ETL/ELT workflows



\## 🏗 Architecture

\- Bronze Layer: Raw data ingestion

\- Silver Layer: Data cleaning \& validation

\- Gold Layer: Analytics-ready datasets



\## ⚙️ Tech Stack

\- Python

\- PySpark

\- Databricks Delta Lake

\- Google Cloud Platform (GCP)

\- SQL
